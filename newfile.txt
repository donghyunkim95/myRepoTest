★ git을 사용하는 순서
    0) 웹브라우저에서 github.com에 sign in 한다.
    1) github 사이트에서 repo 생성
         https://github.com/donghyunkim95?tab=repositories → new
    2) local 컴퓨터 (내컴퓨터)에 repo를 clone한다.ㅏ
       File -> clone repository
    3) 새로운 파일을 생성한다.
    4) commit to main으로 add, commit 한다.
    5) push origin main을 눌러 remote ~에 올린다.
